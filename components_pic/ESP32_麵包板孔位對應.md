## 插入位置
- ESP32 左側針腳：B1 到 B19（共19針）
- ESP32 右側針腳：I1 到 I19（共19針）
- USB 接頭朝向：第 1 排方向（不影響麵包板其他孔位）

## 重要提醒

### 避免使用的針腳
- **GPIO 0**：開機模式選擇（BOOT按鈕）
- **GPIO 1, 3**：UART（上傳程式用，避免占用）
- **GPIO 6-11**：Flash連接（避免使用，可能導致開機失敗）

### 電源針腳位置
- **5V**：B1（直接來自USB，較大電流）
- **3.3V**：B19（最多輸出約600mA）
- **GND**：B6, I13, I19（多個接地點方便布線）

## 測試階段可用針腳

### 按鈕輸入（推薦）
為了避免與PRD規劃衝突，測試階段建議使用：
- GPIO 13 (B5)
- GPIO 15 (I4)
- GPIO 4 (I7)
- GPIO 2 (I5)
- GPIO 23 (I18)

### LED控制輸出（推薦）
- GPIO 5 (I10) - PRD規劃為WS2812控制，現在也用來測試LED

## 未來擴充保留區域

### 音頻模組預留（中間區域）
- B11: GPIO 25 → I2S BCLK
- B10: GPIO 26 → I2S LRC  
- I17: GPIO 22 → I2S DIN（註：GPIO 22也是SCL）
- 需要 5V (B1) 和 GND (B6/I13/I19)
- **建議插在 E9-E12 或 F9-F12 區域**

### WS2812 LED燈條預留（右側中間）
- I10: GPIO 5 → LED DIN
- 需要 5V (B1) 和 GND (I13/I19)
- **建議燈條實體放在麵包板外側或上方**

### 按鈕模組區域（建議放置）
- **5個按鈕模組建議放在 C25-E40 區域**
- 這區域離ESP32有一定距離，避免擁擠
- 可以使用跳線連接到ESP32的GPIO

## 麵包板分區規劃建議

```
      1     5    10    15    20    25    30    35    40    45    50    55    60
     ┌────────────────────────────────────────────────────────────────────────┐
  +  │ ═════════════════════════════════════════════════════════════════ (電源) │
  A  │                                                                          │
  B  │ [ESP32左側針腳 B1-B19]                                                   │
  C  │                                     [測試區]        [按鈕1]  [按鈕2]      │
  D  │                                     [LED區]        [按鈕3]  [按鈕4]      │
  E  │                    [音頻模組]                      [按鈕5]               │
  ─  │ ─────────────────────────────────────────────────────────────────       │
  F  │                    [預留區域]                                            │
  G  │                                                                          │
  H  │                                                                          │
  I  │ [ESP32右側針腳 I1-I19]                                                   │
  J  │                                                                          │
  -  │ ═════════════════════════════════════════════════════════════════ (接地) │
     └────────────────────────────────────────────────────────────────────────┘
```

### 區域說明
- **B1-B19, I1-I19**：ESP32開發板主體
- **C20-E24, F20-F24**：測試LED和元件區域
- **C25-E40**：5個按鈕模組放置區
- **E7-E11, F7-F11**：音頻模組預留區（未來使用）
- **頂部+軌**：連接5V電源軌（從B1接出）
- **底部-軌**：連接GND地線（從B6/I13/I19接出）
